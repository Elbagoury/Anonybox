### Build:
For server side:
> cd Server && chmod +x compile.sh && ./compile.sh

For client side:
> cd Client && chmod +x compile.sh && ./compile.sh

### Running:
For server side:
> cd Server && java main [Encryption key]

For client side:
> cd Client && java client

### Encryption info:

*AES-256 CBC PKCS#5 with IV*

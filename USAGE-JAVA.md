##  Build
### Server
```bash
chmod +x compile.sh
./compile.sh
```
### Client
```bash
chmod +x compile.sh
./compile.sh
```

## Run
### Server
```bash
java main [Encryption key]
```
### Client
```bash
java client
```

## How to use?
When connecting to the server, Enter the IP of the server, the encryption key that is previously known !<br>
Then type ``` help ``` to see available commands !
